Criando Tabelas

Usamos os seguintes elementos para criar uma tabela:

  • A tag Table para criar a tabela
  • A tag tr (table row) para criar uma linha na tabela
  • A tag th para o cabeçalho.

Um esquema básico de tabela é feito assim:
Tudo é circundado por uma tag table, que é a tag que cria a tabela.
Em seguida criamos as linhas, usando a tag tr.
Para o cabeçalho da tabela, entre as tags tr, usamos a tag th
Quando queremos adicionar uma linha, outra ag tr e dentro dessa linha, usamos a tag td, respeitando a hierarquia acima.

Organizando tabelas

Colocaremos o nosso cabeçalho dentro da tag thead.
O nosso corpo, dentro da tag tbody.
E o rodapé, dentro da tag tfoot.
Por fim, usamos a tad caption para descrever sobre o que trata a nossa tabela.
Todos esses elementos facilitam a leitura da tela para pessoas que precisam de alguma acessibilidade!

Tabela complexa

Faremos uma tabela que contém 2 lojas, onde estaremos vendo quantos produtos foram vendidos e produzidos e agruparemos por nome dos produtos.

Thead complexa

Iniciamos criando a tabela. Na tag caption colocamos a nossa descrição/titulo para acessibilidade.
Criamos o cabeçalho com a tag thead, abrimos tr dentro para linha e colocamos dentro desse tr 3 tags th, sendo a primeira vazia.
O atributo rolspan="2" serve para que o elemento ocupe duas linhas, empurrando o restante para o lado.
O atributo colspan="2" tem a mesma função que o rolspan, mas age nas colunas.

Prouzidos x Vendidos por loja
Afonso pena Antonia Pereira
Produzidos Vendidos Produzidos Vendidos

Tbody complexo

Dando continuidade a nossa tabela, iremos criar o tbody, abrir uma linha com o tr, colocar o nome do produto, porém ele será um th, ou seja um cabeçalho, e com a ajuda do emmet criaremos 4 td, para colocarmos o tanto de produtos produzidos e vendidos, e repetiremos, mas para outro produto.
